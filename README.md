# OYNA Home · FamilyHub

Ambient / display-поверхность OYNA для семьи: планшет → настенная панель → hub-устройство.
Всегда-включённый семейный экран, который показывает жизнь ребёнка в OYNA родителю — без превращения в task-manager.

> **Отдельный репозиторий, общая правда данных.** FamilyHub — это отдельная сборка того же продукта OYNA. Один профиль ребёнка, один Supabase, одна auth (STRAT-008). FamilyHub — **read-mostly клиент**, а не второй источник данных.

**Перед любым кодом читай [`CONTRACT.md`](./CONTRACT.md) — контракт изоляции. Он не обсуждается.**

---

## Что это и что это НЕ

**Это:**
- Ambient-экран на планшете/стене, landscape-first, always-on.
- Тот же продукт OYNA, другой форм-фактор и layout.
- Витрина петли познания ребёнка для родителя (co-pilot, не контролёр).

**Это НЕ:**
- Не форк продуктовой логики телефона.
- Не источник схемы БД (ноль миграций отсюда — см. CONTRACT §1).
- Не место монетизации (RevenueCat остаётся на телефоне).
- Не второй auth-контур.

---

## Отношение к основному приложению OYNA

| | Основное приложение (телефон) | FamilyHub (этот репо) |
|---|---|---|
| Кодбейз | основной OYNA-репо | отдельный |
| Bundle id | (основной) | `com.oyna.hub` |
| Данные | владелец схемы, миграции | **read-mostly**, ноль миграций |
| Auth | родитель + ребёнок | родитель / device-scoped сессия |
| Монетизация | RevenueCat | нет |
| Общий код | — | через опубликованный Dart-пакет, read-only |

Изменение общего контракта данных **всегда** инициируется из основного репо, версионируется в пакете, затем подтягивается сюда. Основной `lib/` из этого репо не редактируется никогда.

---

## Структура (целевая)

```
lib/
  main.dart                 # entry, deviceMode: hub
  core/
    supabase/               # обёртка клиента (anon key + RLS, read-mostly)
    theme/                  # токены (emerald ~#0E7A40 / #159A55, не gold)
    models/                 # тонкие копии моделей ИЛИ импорт общего пакета
  features/
    ambient/                # always-on главный экран
    child_pulse/            # что происходит с ребёнком (петля познания, view-only)
    screensaver/            # заставка / idle-режим
    kiosk/                  # lockdown-режим (ПОСЛЕ BET-001, гейт железа)
```

## Стек

Flutter (Riverpod, GoRouter) · Supabase (тот же PROD-проект, anon + RLS) · без RevenueCat.
Бренд: emerald `#0E7A40` (GDARK) / `#159A55` — не gold. Фоны — сплошные цветовые блоки, не градиенты.

## Секреты

Все ключи — в env / Supabase Edge secrets, **не в git** (ARCH-RULE-001). Anon key + RLS.
Запрос `service_role` — стоп-сигнал.
